# Files

* `clean_cdc_birth_data.csv`: USA number of births per day 1994-2003 by Centers for Disease Control and Prevention
* `US-2000-2014-SSA.txt`: USA number of births per day 2000-2014 by Social Security Administration
* Data for 1969-1988 is also included in the `birthdays/data` directory of the GitHub repository since
it is used in that case study.

For the CDC data see: 

* 1994:    https://www.cdc.gov/nchs/data/statab/t941x16.pdf
* 1995:    https://www.cdc.gov/nchs/data/statab/natfinal1995annvol1_16.pdf
* 1996:    https://www.cdc.gov/nchs/data/statab/natfinal1996.annvol1_16.pdf
* 1997:    https://www.cdc.gov/nchs/data/statab/t1x1697.pdf
* 1998:    https://www.cdc.gov/nchs/data/statab/t981x16.pdf
* 1999:    https://www.cdc.gov/nchs/data/statab/t991x16.pdf
* 2000:    https://www.cdc.gov/nchs/data/statab/t001x16.pdf
* 2001:    https://www.cdc.gov/nchs/data/statab/natfinal2001.annvol1_16.pdf
* 2002:    https://www.cdc.gov/nchs/data/statab/natfinal2002.annvol1_16.pdf
* 2003:    https://www.cdc.gov/nchs/data/statab/natfinal2003.annvol1_16.pdf
